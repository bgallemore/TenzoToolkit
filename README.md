# TenzoToolkit

Institutional culinary operations toolkit for healthcare, assisted living, memory care, and batch-production kitchens.

TenzoToolkit combines practical kitchen documentation, Python command-line tools, CSV data workflows, and future analytics expansion into a portfolio-ready operations project.

## Current Modules

- HACCP documentation repository
- Kitchen Prep Calculator CLI
- Recipe Cost Calculator

## Project Goals

- Model authentic institutional culinary workflows
- Demonstrate Python, Git/GitHub, Markdown, and CSV skills
- Support assisted living and memory care kitchen operations
- Build toward SQL, Power BI, dashboards, and AI-assisted planning
